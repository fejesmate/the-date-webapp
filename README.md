# the-date-webapp
How to start the project:
- install NodeJS
  - Linux
  ```
  apt-get install nodejs
  ```
  - Windows
  ```
  https://nodejs.org/en/download/
  ```
- Checkout project
  ```
  git checkout https://github.com/szdani/the-date-webapp
  ```
- Open project dir, and install node modules
 ```  
 npm install
 ```
- Start project
  ```
  npm start
  ```
